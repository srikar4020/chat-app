# Core Features
- Real-time chat using WebSockets
- One-to-one private messaging
- Persistent chat history stored in database
- User authentication (Login / Signup)
- Responsive UI (Mobile + Desktop)


### Installation & Setup (Local)
  1. Clone Repository
  2.git clone <repo-link>
  3.npm install
  4.cd client
  5.npm run dev
---

# How It Works

1. User logs in using authentication system  
2. Frontend connects to backend via Socket.io  
3. Messages are sent and received instantly  
4. Messages are stored in MongoDB  
5. UI updates in real-time
